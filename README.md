Starbucks Food Review Analysis 

This project analyzes Starbucks food reviews using sentiment analysis and data visualization techniques to gain insights into customer satisfaction.

📌 Features:

Dataset: Reviews collected from Kaggle
EDA: Data cleaning, visualization, and exploratory data analysis
Sentiment Analysis: Implemented using VADER to classify reviews as Positive, Neutral, or Negative
Visualization: Insights presented using Matplotlib, Seaborn, and Plotly
Key Findings: Identified common customer concerns and seasonal trends in ratings
🚀 Tech Stack: Python, Pandas, Seaborn, Matplotlib, Plotly, NLTK

🔍 Use Cases:

Understanding customer sentiment on Starbucks food offerings
Identifying areas for improvement
Leveraging seasonal trends for business strategy
